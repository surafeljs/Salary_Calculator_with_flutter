# Salary Calculator with Flutter 💰📱

A simple **Salary Calculator mobile application** built using **Flutter** and **Dart**.
This app helps users quickly calculate salary details such as **gross salary, deductions, and net salary**.

---

## 🚀 Features

* 📊 Calculate **Net Salary**
* ➕ Add **allowances and bonuses**
* ➖ Calculate **deductions**
* ⚡ Fast and simple user interface
* 📱 Works on **Android and iOS**
* 🎨 Clean Flutter UI

---

## 🛠️ Built With

* **Flutter**
* **Dart**
* **Material Design**

## 📦 Installation

1. Clone the repository

```
git clone https://github.com/surafeljs/Salart_Calculator.git
```

2. Go to the project directory

```
cd Salart_Calculator
```

3. Get dependencies

```
flutter pub get
```

4. Run the application

```
flutter run
```

---

---

## ⚙️ How It Works

1. Enter your **basic salary**
2. Add **allowances or bonuses**
3. Add **deduction values**
4. The app calculates the **final net salary automatically**
